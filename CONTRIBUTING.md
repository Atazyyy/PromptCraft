# Contributing to PromptCraft

🎨 Terima kasih atas kontribusi kamu!

## Cara Berkontribusi

### 1. Fork & Clone

```bash
# Fork repo terlebih dahulu di GitHub
# Kemudian clone fork kamu
git clone https://github.com/YOUR-USERNAME/PromptCraft.git
cd PromptCraft
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Buat Branch Baru

```bash
git checkout -b feature/nama-feature
# atau
git checkout -b fix/nama-bug
```

### 4. Develop

```bash
npm run dev
```

Buka [http://localhost:5173](http://localhost:5173) untuk melihat perubahan.

### 5. Commit & Push

```bash
git add .
git commit -m "Add: deskripsi perubahan"
git push origin feature/nama-feature
```

### 6. Buat Pull Request

1. Buka repo asli di GitHub
2. Klik "New Pull Request"
3. Pilih branch kamu
4. Tambahkan deskripsi perubahan
5. Submit!

## 📝 Panduan Kontribusi

### Lessons & Quiz
- Ikuti format yang sudah ada di `lessons.ts` dan `quizzes.ts`
- Pastikan konten accurate dan helpful
- Test quiz questions sebelum push

### Templates
- Template harus practical dan usable
- Ikuti struktur yang sudah ada
- Tambahkan contoh yang relevan

### UI/UX
- Ikuti design system yang ada (Tailwind)
- Test responsive design
- Pastikan accessibility

### Bug Fixes
- Jelaskan bug yang di-fix
- Sertakan steps to reproduce
- Test sebelum push
